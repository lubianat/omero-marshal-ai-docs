Release Notes
=============

This project ships documentation-only releases. Upstream ``omero-marshal`` code
and behaviour are authoritative; always confirm against the library and its
tests.

0.9.2 (2025-11-27)
------------------

- Retargeted wording toward engineers integrating OMERO, not just bioinformatics users.
- Updated README, index, and overview to reflect the broader audience.

0.9.1 (2025-11-27)
------------------

- Clarified quickstart dependency installs, JSON handling, and RTD downloads.
- Glossary refreshed with examples and cross-links into usage/tutorials.
- Added a design notes page for codec internals, patterns, and schema handling.
- Added an IDR read-only tutorial for real-world round-trips.
- Documented basic test-running guidance and codec registration notes.
- Expanded troubleshooting/FAQ and clarified the dynamic registry behaviour.
- Enabled RTD downloadable artifacts (PDF, EPUB, HTML zip).

See also: ``CHANGELOG.md`` in the repository root for a repository-level
summary.
